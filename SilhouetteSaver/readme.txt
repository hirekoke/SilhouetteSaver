Silhouette Saver

  動画を透過させて再生するスクリーンセーバです。
  最低限の機能しか実装していないので期待しないでください。
  
  !! 動作は無保証です。どうしようも無くなったらタスクマネージャから落として下さい!!!
  また動作確認がまだ不十分なので、動作しない、落ちた等があれば、
  実行環境と共にご連絡いただければ、対応するかもしれません。
  またソースも公開しているので、ご自分で対応していただけると小躍りします。

  内容物:
    - SilhouetteSaver.scr: スクリーンセーバ本体
    - SilhouetteSaver.dll: 実行に必要なライブラリ
    - readme.txt: このファイル

  動作環境:
    - Windows Vista/7 (XP は動作確認できていません)
    - Microsoft .NET Framework 4
    - Direct X ランタイム

  インストール:
    - SilhouetteSaver.scr を右クリックし、メニューから「インストール」を選択。

  アンインストール:
    - 内容物を削除。
    - 設定ファイルが <ユーザディレクトリ>\AppData\Roaming\SilhouetteSaver\ 以下にあるので
      ディレクトリごと削除。
	  (XP の場合は <ユーザディレクトリ>\Application Data\SilhouetteSaver\)

  設定:
    - 再生リスト
      * パス: 再生する動画ファイル。ダブルクリックで動画を選択。
              基本的に Windows Media Player で再生できる動画は再生可能なはずです。
      * 音量: その動画を再生する際の音量。0 で無音。
      * 効果: 再生する際の効果。ダブルクリックで編集。
      * テスト: ボタンを押すと設定された内容でその動画だけを再生。

      * 追加: 空行を選択し、パスを設定。
              新しく追加した後は他の行を選択するか、適用ボタンをクリックすることで
              新たな空行が追加されます。
      * 順番の入れ替え: 行の何もない部分(テストの右側)ドラッグ。
      * 削除: 行を選択し、左下の「削除」ボタンクリック。

   - 効果
      * 灰み: 色をどのくらい残さないか。1.0で完全な白黒。
              色を残すと透明度が落ちるので注意。
      * 明るさ: 明るさの調整。効果のほどは不明。
      * 反転: 白黒反転。

  把握しているバグ:
    - スクリーンセーバ起動後、少しマウスを動かさないとカーソルが消えない
      (スクリーンセーバとしては致命的)
    - タブレット使用時に設定を行うと落ちることがある
    - プレイリストで無駄に空行ができる
